Teste de Flexbox

Flexbox (ou Flexible Box Layout) é um modelo de layout do CSS3 que facilita a criação de layouts flexíveis e responsivos. Ele permite alinhar e distribuir espaço entre os elementos dentro de um container, mesmo quando o tamanho dos elementos é desconhecido ou dinâmico. O Flexbox funciona a partir de um container (elemento pai) que usa display: flex. Os elementos filhos desse container se tornam "itens flexíveis".
